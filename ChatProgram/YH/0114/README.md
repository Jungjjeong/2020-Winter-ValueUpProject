<br/>
 
> ## How to run<br/>
<pre>
# Kurento 환경 세팅.pdf 참조
git clone https://github.com/CSID-DGU/2020-2-OSSP1-WebRTC-6.git
cd 2020-2-OSSP1-WebRTC-6
npm install kurento-client
npm install
npm start
</pre></br>
